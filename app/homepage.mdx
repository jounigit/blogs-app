## Welcome to Blog app

This is **a modern blogging** platform built with Nextjs.

### Features

 - **Create and Manage Blogs** - Write and publish blogs
 - **User Profiles** - User page
 - **Authentication** - Secure login and registration
 - **Comments** - Get comments

### Getting Started

To start:

 1. [Register](https://google.com) for new account
 2. [Login](https://google.com) to access dashboard
 3. [Create a new blog](https://google.com)

---

Build using Nextjs, React and Drizzle ORM.
